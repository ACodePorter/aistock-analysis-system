# 技术分析报告口语化摘要 — 提示词模板

| 字段 | 值 |
|------|-----|
| **version** | v1.0 |
| **用途** | 将专业的股市技术分析摘要转化为普通投资者可读的简明解读 |
| **调用方** | `backend/app/reports/report.py` → `llm_summarize()` |
| **模型要求** | GPT-4o / GPT-5-mini |
| **max_output_tokens** | 1024 |

---

## User Prompt Template

```
把下面的股市技术分析摘要转成口语化、给普通投资者看的简明解读，保留关键数字，控制在150字以内，避免夸大。

{text}
```

## 输出格式

纯文本，不超过 150 字的口语化解读。

## 关键约束

- 保留关键数字（价格、涨跌幅、指标值等）
- 口语化表达，面向普通投资者而非专业人士
- 控制在 150 字以内
- 避免夸大或过度渲染
- 不使用专业术语缩写（如需提及应附带解释）

## 输入示例

```
技术分析摘要：沪指收于3245.67点，涨1.2%，站上MA20均线（3220），RSI(14)=62.3处于中性偏强区间，MACD金叉确认，成交量放大至8500亿，北向资金净流入45亿。短期支撑位3200，阻力位3280。
```

## 输出示例

```
今天大盘涨了1.2%收在3245点，站稳了20日均线，买卖力量偏向多头。成交8500亿比之前活跃，北向资金也净买了45亿，说明外资看好。短期3200点是支撑，3280点有压力。整体走势还不错，但别追高，注意3280能不能突破。
```
