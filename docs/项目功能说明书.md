# 🚀 AI股票分析系统 - 项目功能说明书

## 📋 项目概述

**AI股票分析系统 (AI Stock Analysis System)** 是一个全功能的智能股票分析与新闻监控平台，集成了机器学习预测、技术指标分析、智能新闻收集、情感分析等多项前沿技术。系统专为中国A股市场设计，提供从数据采集到智能分析的完整解决方案。

### 🎯 核心价值
- **🤖 AI驱动预测**：基于神经网络的多步股价预测
- **📰 智能新闻监控**：自动化新闻收集与情感分析  
- **📊 实时技术分析**：20+技术指标实时计算
- **🔄 全自动化运行**：无人值守的数据管道
- **💡 投资决策支持**：综合分析报告生成

## 🏗️ 系统架构

```
前端组件架构:
├── Dashboard.tsx          # 主仪表板界面
```

**核心功能**:
- 📊 **实时仪表板**: 股票价格、预测、技术指标可视化
- 📰 **新闻管理中心**: 新闻搜索、过滤、情感分析展示
- 📈 **图表可视化**: Recharts集成的交互式图表
- 🎛️ **股票监控**: 自选股管理和批量操作

#### 📰 新闻管理中心（前端）

提供面向资讯的一站式管理与分析能力，支持统计概览、条件筛选、列表管理与批量操作，并与智能采集任务联动。

- 概览统计
  - 指标：`总文章数`、`今日文章数`
  - 情感分布：`积极`、`消极`、`中性`（百分比）
  - 排行：`热门来源 Top10`、`热门股票 Top10`

- 列表与筛选
  - 条件：关键字（前端过滤）、时间范围、来源、`类别`、`情感类型`、`关联股票`
  - 分页：`limit`（默认50, 1-200）、`offset`（默认0）
  - 字段：标题、摘要、来源、发布时间、情感类型/分值、相关股票标签、原文链接

- 操作与批量
  - 单条：`收藏/取消收藏`、`标记已读/未读`
  - 批量：对勾选记录执行`收藏/取消收藏`、`标记已读/未读`
  - 快捷：`隐藏/显示统计`、`刷新`

- 智能采集联动
  - 一键执行：`智能采集`（基于策略与自选股）、`每日采集`
  - 查看：采集执行状态与任务进度

- 接口映射（后端）
  - 统计/枚举
    - `GET /api/news/stats` 统计卡片
    - `GET /api/news/sources` 来源列表
    - `GET /api/news/categories` 类别枚举
    - `GET /api/news/sentiment-types` 情感类型枚举
  - 列表/过滤
    - `GET /api/news/articles?category&sentiment&symbol&limit&offset`
  - 单条/批量操作
    - `POST /api/news/{id}/bookmark` 切换书签
    - `POST /api/news/{id}/read` 切换已读
    - `POST /api/news/batch-update?action=bookmark|unbookmark|mark_read|mark_unread`
    - `DELETE /api/news/{id}` 删除
  - 采集与策略
    - `POST /api/news/collect/intelligent` 智能采集
    - `POST /api/news/collect/daily` 每日采集
    - `GET /api/news/collection/status` 采集状态
    - `GET /api/news/strategies` 策略列表
    - `POST /api/news/strategies/execute/{name}` 执行策略
  - 个股情感
    - `GET /api/news/sentiment/{symbol}?days=7` 情感趋势与摘要
  - MongoDB存档（可选）
    - `GET /api/storage/stock-news/{stock_symbol}` 股票历史新闻
    - `GET /api/storage/stock-statistics/{stock_symbol}` 存档统计
    - `POST /api/storage/cleanup?days_to_keep=90` 清理旧数据

提示：若界面出现异常日期（如 1970-01-01），通常是 `published_at` 缺失；需要在采集/解析阶段完善时间字段。
